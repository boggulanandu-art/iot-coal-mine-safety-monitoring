# ⛏️ IoT-Based Coal Mine Safety Monitoring with Hazard Detection

## 📌 Project Overview

Coal mining is one of the most hazardous industries due to the presence of toxic gases, high temperatures, and unsafe environmental conditions. This project presents an IoT-based safety monitoring system that continuously observes critical mine parameters and provides real-time alerts whenever dangerous conditions are detected.

The system helps improve worker safety, reduce accidents, and enable remote monitoring through IoT technology.

---

## 🎯 Objectives

* Monitor environmental conditions inside coal mines.
* Detect hazardous gas leakage at an early stage.
* Measure temperature variations in real time.
* Generate instant alerts during unsafe conditions.
* Improve overall mine safety using IoT technology.

## ⚙️ Working Principle

1. Environmental sensors continuously collect data from the mine.
2. Arduino processes the sensor readings.
3. Data is transmitted to an IoT platform through the ESP8266 module.
4. Threshold values are continuously monitored.
5. When dangerous conditions are detected, alerts are triggered immediately.
6. Supervisors can monitor conditions remotely through the cloud platform.
